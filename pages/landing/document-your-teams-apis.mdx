### Document an API using traffic

Capture API traffic from your browser, tests or REST client.
```shell
optic oas capture openapi.yml https://{your-api}
```

Optic adds undocumented endpoints to your OpenAPI spec.
```shell
optic oas verify openapi.yml --document all
```

**[Read Documentation â†’](/docs/openapi/generate-from-traffic)**

### Know when an APIs changes

Start tracking changes to your team's API by adding them to Optic Cloud:

```shell
optic login
optic api add openapi.yml
```
